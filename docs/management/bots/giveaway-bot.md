# GiveawayBot

We use this bot to host giveaways in the server. 

https://giveawaybot.party/

| Prefix | Website                    |
| ------ | -------------------------- |
| `g!`   | https://giveawaybot.party/ |

```
/gcreate
```

Use this command in the channel you want to host the giveaway in.
